# Description
Wordpress plugin builder framework

## Install

1. Download and create new project
```
composer create-project codexshaper/wpb-framework project-name
```
2. Autoload
```
composer dump-autolaod
```
3. Install npm
```
npm install
```
4. Build or Serve
```
npm run build
```
OR
```
npm run serve
```
