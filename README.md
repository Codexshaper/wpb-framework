# Description
Wordpress plugin builder framework

## Install

1. Download and create new project
```
composer create-project codexshaper/wpb-framework plugin-name
```
2. Autoload
```
composer dump-autoload
```
3. Install npm
```
npm install
```
4. Build or Serve
```
npm run build
```
OR
```
npm run serve
```
